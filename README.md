# Conditional_Independence
The MATLAB codes for implementing simulations in the paper "Limit theorems of Azadkia-Chatterjee's conditional graph correlation" (by Muhong Gao, Fang Han, and Qizhai Li)
1. The main script files "Sim_ECP.m" and "Sim_power.m" are used for reproducing simulation results in Section 5.1 and Section 5.2 respectively. When using the program, user needs to follow the prompts in the command window to select the required information, such as model number and replication times.
2. The folder "Functions/" include the functions required for implementing the program.
3. The folder “Results/” includes the data files containing the true values of T and $\sigma^2$, as well as the corresponding result files produced by the program.
